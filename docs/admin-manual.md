## 1. 入室方法

講師専用URLをブラウザ（Chrome推奨）で開いてください。

| 講師 | URL |
|------|-----|
| 元沢信昭 | 運営から別途お知らせするURL |
| 塚越暁 | 運営から別途お知らせするURL |

URLを開くと**名前入力をスキップして自動的にメインルームへ入室**します。

> 警告: 講師URLは部外者に共有しないでください。漏洩した場合は運営にお知らせください（キーを再発行します）。

## 2. 画面の見方

講師の画面は **メインエリア**（左）と **ダッシュボード**（右）の2カラム構成です。

### メインエリア（左）

```
┌────────────────────────────────────┐
│ メインルーム          [自分の名前] │  ← ヘッダー（現在のルーム名）
├────────────────────────────────────┤
│                                    │
│  [受講生A の画面]  [受講生B の画面]│  ← 参加者グリッド
│                                    │     クリックで拡大
│  [受講生C の画面]  [受講生D の画面]│
│                                    │
├────────────────────────────────────┤
│  🎤 マイク  📷 カメラ  🖥️ 画面共有  │  ← コントロールバー
└────────────────────────────────────┘
```

### 講師ダッシュボード（右パネル）

```
┌──────────────────────┐
│ 講師ダッシュボード   │
│ [自分の名前]         │
├──────────────────────┤
│ ✋ 挙手中 (1)         │  ← 挙手している受講生
│  田中さん            │
│  [BO1へ][BO2へ][BO3へ]│
├──────────────────────┤
│ 参加者 (5)           │  ← 全受講生リスト
│  ● 佐藤さん          │
│    [BO1へ][BO2へ][BO3へ]
│  ● 鈴木さん          │
├──────────────────────┤
│ ブレイクアウト状況   │
│  BO1: 使用中         │
│  BO2: -              │
│  BO3: -              │
└──────────────────────┘
```

## 3. 受講生の管理

### 受講生をブレイクアウトへ移動する

1. 右パネルの「参加者」リストで対象の受講生を見つける
2. `[BO1へ]` `[BO2へ]` `[BO3へ]` のいずれかをクリック
3. **対象受講生だけがBOへ移動します**（講師はメインルームに残ります）

### 受講生の画面を拡大して確認する

1. メインエリアの参加者グリッドで、確認したい受講生のサムネをクリック
2. 拡大表示に切り替わる
3. もう一度クリック、または「グリッドに戻る」ボタンで元に戻る

### 挙手中の受講生を優先対応する

右パネル上部の「✋ 挙手中」セクションに挙手中の受講生が古い順に並びます。最上位の受講生から対応してください。

## 4. ブレイクアウトの開始・戻り

### 開始する

1. 右パネルで対象受講生の `[BO○へ]` をクリック
2. 対象受講生だけが自動的にそのBOへ移動（講師はメインルームに残ります）

### BO内での操作

- 通常通りマイク・カメラ・画面共有が使えます
- 他の受講生がBOに聴講参加してくる場合があります（発言も可能）
- BOへ移動した受講生は、自分のタイミングで「メインに戻る」から復帰できます

### メインへ戻る

講師側からBO内の全員を一括でメインへ戻す操作はありません。指導や確認が終わったら、受講生本人にコントロールバーの **「メインに戻る」** を押してもらってください。

### 講師2名で並行指導する場合

- BO状況を確認しながら、同じBOに受講生が重なりすぎないように割り当ててください
- 参加者本人は任意のタイミングでメインへ戻れます

---

## 5. 挙手への対応

1. 右パネルの「✋ 挙手中」に受講生名が表示される
2. 対応する受講生の `[BO○へ]` をクリック
3. BOに移動すると挙手フラグが自動的に解除される

挙手フラグは以下のタイミングで解除されます:
- 受講生自身が挙手ボタンを再度押したとき
- 講師がその受講生をBOへ移動させたとき

## 6. よくあるトラブル

| 症状 | 原因 | 対処 |
|------|------|------|
| 受講生の画面が表示されない | 画面共有していない | 受講生に画面共有を促す |
| 移動ボタンを押しても動かない | ネットワーク遅延 | 少し待ってから再試行 |
| 音声が聞こえない | マイクがOFFまたは許可なし | コントロールバーのマイクボタンを確認 |
| 講師URLが使えない | キーが古い | 運営にキーの再発行を依頼 |
| BOが3部屋とも埋まっている | 同時使用 | 現在進行中のBOが終わるまで待つ |

## 操作チートシート

| 操作 | 方法 |
|------|------|
| 入室 | 講師専用URLを開く（自動入室） |
| 受講生をBOへ移動 | 右パネル → 受講生名 → `[BO○へ]` |
| 挙手に対応 | 右パネル「挙手中」→ `[BO○へ]` |
| 受講生の画面を拡大 | グリッドのサムネをクリック |
| BOから戻る | 受講生本人がコントロールバー「メインに戻る」 |
| マイクON/OFF | コントロールバー「🎤」 |
| 画面共有 | コントロールバー「🖥️」 |
