## 1. はじめて参加するとき

### ステップ1: URLを開く

講師からDiscordで送られてきたURLをクリックして、ブラウザ（Chrome推奨）で開きます。

### ステップ2: 名前を入力する

「お名前を入力してください」という画面が表示されます。

1. テキストボックスに**自分の名前**を入力（20文字以内）
2. 「**参加する**」ボタンをクリック

### ステップ3: ブラウザの許可を承認する

初回のみ、以下の許可を求める画面が表示されます。

| 許可の種類 | 説明 | 選択 |
|-----------|------|------|
| マイク | 声を送るために必要 | **許可** |
| カメラ | 映像を送るために必要（任意） | 許可 または ブロック |
| 画面共有 | 作業画面を見せるために必要 | 画面共有ボタンを押したときに確認 |

> ✅ マイクは「許可」を選んでください。カメラはご自身の判断で構いません。

「ブロック」を選んでしまった場合は → [7. 困ったときは](#7-困ったときは) を参照

### ステップ4: 自習室に入る

名前入力と許可が完了すると、**メインルーム**に入室します。他の参加者の画面が表示されます。

## 2. 自習室の画面の見方

```
┌─────────────────────────────────────────┐
│ メインルーム                  [名前]    │  ← 現在いる部屋の名前
├─────────────────────────────────────────┤
│                                         │
│  [Aさんの画面]    [Bさんの画面]         │  ← みんなの画面共有
│                                         │     クリックすると大きく見える
│  [Cさんの画面]    [Dさんの画面]         │
│                                         │
├─────────────────────────────────────────┤
│ ブレイクアウトルーム                    │  ← 個別指導中の部屋の情報
│ [BO1に聴講参加] [BO2に聴講参加] ...   │
├─────────────────────────────────────────┤
│ 🎤 マイク  📷 カメラ  🖥️ 画面共有  ✋ 挙手 │  ← 操作ボタン
└─────────────────────────────────────────┘
```

### 他の人の画面を大きく見たい

参加者グリッドの中の見たい画面をクリックすると、大きく表示されます。もう一度クリックするか「グリッドに戻る」ボタンで元に戻ります。

## 3. マイク・カメラ・画面共有の操作

画面下部のコントロールバーで操作します。

| ボタン | 役割 | 押すと |
|--------|------|--------|
| 🎤 マイク | 自分の声を送る | ON ↔ OFF を切替 |
| 📷 カメラ | 自分の映像を送る | ON ↔ OFF を切替 |
| 🖥️ 画面共有 | 自分の作業画面を見せる | 開始 ↔ 停止を切替 |

### 画面共有のやり方

1. 「🖥️ 画面共有」ボタンをクリック
2. ブラウザが「何を共有しますか？」と聞いてきます
3. **「画面全体」**または**「ウィンドウ」**を選んで「共有」をクリック
4. 自分の画面が他の参加者に見えるようになります

> 💡 作業しているアプリのウィンドウだけを選ぶと、他の画面が見られる心配がありません。

### スマートフォンをお使いの場合

スマートフォンからは**聴講のみ可能**です。画面共有や発言はPCからお願いします。

## 4. 質問したいとき（挙手）

質問があるときは「挙手」機能を使います。

### 挙手の方法

1. 画面下部の **「✋ 挙手」** ボタンをクリック
2. ボタンがオレンジ色に変わり「**挙手中**」と表示されます
3. 講師に通知が届きます → 少し待ってください

### 挙手を取り消すとき

もう一度「✋ 挙手中」ボタンをクリックすると、挙手が取り消されます。

### 挙手の後は？

講師が対応できるタイミングになると、**ブレイクアウトルーム（個別指導室）に自動的に移動**します。特に操作は不要です。

---

## 5. ブレイクアウトルーム（個別指導）

### 自動移動について

挙手後、講師が準備できると「BO1（または BO2・BO3）に移動してください」という指示が届き、**自動的に個別指導室に移動します**。特に何かボタンを押す必要はありません。

### ブレイクアウトルームでできること

- 講師と1対1で話せます
- マイク・カメラ・画面共有は通常通り使えます
- 興味があれば他の参加者も聴講に来ることがあります（歓迎です）

### メインルームに戻るとき

コントロールバーの **「🚪 メインに戻る」** ボタンをクリックすると、いつでもメインルームに戻れます。

### 指導が終わったら

講師が「⏹️ 終了してメインへ」を押すと、自動的にメインルームに戻ります。

### 他の人の個別指導を聴講したいとき

メインルームの下部に「BO1に聴講参加」などのボタンが表示されています。クリックすると、そのブレイクアウトルームに参加できます。発言もできます。

戻るときは「🚪 メインに戻る」ボタンを押してください。

---

## 6. 退出するとき

セッション終了時は、ブラウザのタブを閉じるか、ウィンドウを閉じてください。

特別な退出ボタンはありません。閉じるだけでOKです。

---

## 7. 困ったときは

### ブラウザがマイクを「ブロック」してしまった場合

**Chromeの場合:**

1. アドレスバー左側の 🔒 鍵アイコンをクリック
2. 「マイク」の設定を「許可」に変更
3. ページを再読み込み（F5 または Ctrl+R）

**それでも解決しない場合:** ブラウザを閉じて、URLを開き直してください。

---

### 接続が切れてしまった場合

ページを再読み込みしてください（F5 または Ctrl+R）。名前を再入力すると自動的に再接続されます。

---

### 画面共有が始まらない場合

- ブラウザが最新版（Chrome推奨）か確認してください
- スマートフォンからは画面共有できません（PCをご使用ください）
- ブラウザの「画面共有」権限が「許可」になっているか確認してください

---

### 音声が聞こえない・届かない場合

| チェック項目 | 確認方法 |
|-------------|---------|
| 自分のマイクがONか | 🎤 ボタンがグレーでないか確認 |
| 相手のマイクがONか | 相手の名前の隣のアイコンを確認 |
| PCのマイク設定 | PCのサウンド設定でマイクが有効か確認 |
| イヤホン | イヤホンを使うとハウリング（ノイズ）を防げます |
